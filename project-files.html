Project Files¶

The project Files listing gives you an overview of your files in that project.

  • Show Hidden Files
  • Hide Temporary Files
  • Snapshots

Click on a line to open a file, or select one or more files via the checkbox on the left to manipulate them (deleting, moving, …)

files list

Show Hidden Files¶

By convention, a file whose name begins with a dot is a hidden file. It will not show up in various file listings by default. Click the eye icon with a slash () to include hidden files in the list; the icon changes to an eye () without a slash. Click the icon to exclude hidden files.

files list showing hidden files

Hide Temporary Files¶

Processing some programs, notably LaTeX source (e.g. .tex files) causes temporary files to be generated. These can clutter your file listing. Click the mask icon () to toggle display of hidden files on or off.

files list filtered for “latex-sample”, temporary files masked

files list filtered for “latex-sample”, showing temporary files

Snapshots¶

Click on the “ Backup” button to switch to a directory containing consistent point-in-time backups of all your files. Use this in case you have deleted a whole set of files, or just want to restore them from a while ago.

Note: Snapshots are in a read-only file system. If you want to modify a file that resides in a snapshot, you will need to copy it into a writable part of your project first.
